Implement Milestone 29 now.

Goal:
Transform ParsePilot into a recruiter workflow tool by adding:

1. Candidate outreach system (invite to interview / test)
2. Recruiter dashboard UI (clean, fast, decision-focused)
3. Basic CRM pipeline to track candidates

IMPORTANT:
- UI must be clean (Stripe/Notion style)
- Fast decision making is priority
- Avoid clutter (NOT a traditional ATS)

--------------------------------------------------
1. DATABASE MODELS
--------------------------------------------------

Create:

Candidate:
- id
- name
- email
- score
- skills
- experience
- status (new | invited | accepted | rejected)
- createdAt

Invite:
- id
- candidateId
- recruiterId
- type (interview | test)
- status (sent | opened | accepted | rejected)
- scheduledAt
- meetingLink
- message
- createdAt

Recruiter:
- id
- email
- organizationName
