MILESTONE 32 — RESUONE RECRUITER RANKING DASHBOARD

GOAL
Build the recruiter workflow where an institution or recruiter can:
1. Create or paste one job description
2. Upload multiple candidate CVs
3. Parse all CVs
4. Match every candidate against the same job
5. Rank candidates automatically
6. Filter by score, missing skills, seniority, and recommendation
7. Open a candidate comparison view
8. Shortlist / reject / mark for interview
9. Export ranked results later

WHY THIS MILESTONE
This milestone turns Resuone from “single CV analyzer” into an actual recruiter product.

MAIN USER FLOW
1. Recruiter creates a job
2. Recruiter uploads 5–100 CVs
3. System extracts text from every file
4. System parses each CV using Claude
5. System runs candidate-vs-job analysis for each candidate
6. System stores normalized results in Supabase
7. Recruiter sees:
   - ranking table
   - score
   - strengths
   - concerns
   - missing skills
   - interview recommendation
8. Recruiter clicks into a candidate comparison screen
9. Recruiter updates candidate status

STACK
- Next.js App Router
- Supabase Postgres
- Supabase Storage
- Anthropic Claude API
- Zod validation
- Queue-ready async structure
- Recharts or plain table UI for analytics later

NEW DATABASE TABLES

  id uuid primary key default gen_random_uuid(),
  recruiter_user_id uuid not null,
  title text not null,
  company text,
  location text,
  raw_description text not null,
  normalized_requirements jsonb default '{}'::jsonb,

create table if not exists recruiter_candidates (
  id uuid primary key default gen_random_uuid(),
  recruiter_job_id uuid references recruiter_jobs(id) on delete cascade,
  candidate_document_id uuid references candidate_documents(id) on delete set null,
  full_name text,
  email text,
  current_title text,
  current_location text,
  parsed_cv jsonb default '{}'::jsonb,
  status text default 'new',
  created_at timestamptz default now()
);

create table if not exists recruiter_candidate_matches (
  id uuid primary key default gen_random_uuid(),
  recruiter_job_id uuid references recruiter_jobs(id) on delete cascade,
  recruiter_candidate_id uuid references recruiter_candidates(id) on delete cascade,
  overall_score integer not null,
  interview_recommendation text,
  matching_skills jsonb default '[]'::jsonb,
  missing_skills jsonb default '[]'::jsonb,
  strengths jsonb default '[]'::jsonb,
  concerns jsonb default '[]'::jsonb,
  recruiter_summary text,
  model_name text,
  rank_position integer,
  created_at timestamptz default now()
);

create index if not exists idx_recruiter_jobs_user on recruiter_jobs(recruiter_user_id);
create index if not exists idx_recruiter_candidates_job on recruiter_candidates(recruiter_job_id);

BATCH CANDIDATE UPLOAD
For upload batch route:

- extract text from each
- store each file metadata

IMPORTANT
Do NOT do all heavy work in one huge request forever.
Split the flow:
1. upload-batch = upload + store pending records
2. run-batch-analysis = process in smaller chunks

CHUNKING RULE
Process candidates in chunks of 5 first.
This is safer for Replit and API stability.

RUN-BATCH-ANALYSIS ROUTE
Input:
{
  "recruiterJobId": "...",
  "candidateIds": ["..."]
}

For each candidate:
1. get parsed_cv
2. get recruiter_jobs.normalized_requirements
3. build match prompt
4. call Claude
5. validate JSON
6. insert recruiter_candidate_matches row

RANKING LOGIC
After all matches exist, run rank route:
- sort by overall_score descending
- second sort by recommendation priority:
  strong_yes > yes > maybe > no
- third sort by missing_skills count ascending
- assign rank_position

Add helper:
lib/ai/scoring.ts

export function recommendationWeight(value: string) {
  if (value === "strong_yes") return 4;
  if (value === "yes") return 3;
  if (value === "maybe") return 2;
  return 1;
}

export function rankCandidates(matches: any[]) {
  return matches
    .sort((a, b) => {
      if (b.overall_score !== a.overall_score) {
        return b.overall_score - a.overall_score;
      }

      const recDiff =
        recommendationWeight(b.interview_recommendation) -
        recommendationWeight(a.interview_recommendation);

      if (recDiff !== 0) return recDiff;

      return (a.missing_skills?.length || 0) - (b.missing_skills?.length || 0);
    })
    .map((item, index) => ({
      ...item,
      rank_position: index + 1
    }));
}
