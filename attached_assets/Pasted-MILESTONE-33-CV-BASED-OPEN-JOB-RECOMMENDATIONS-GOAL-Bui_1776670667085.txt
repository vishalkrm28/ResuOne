MILESTONE 33 — CV-BASED OPEN JOB RECOMMENDATIONS

GOAL
Build a user-facing feature where a candidate uploads their CV and gets recommended real open jobs that match their experience, skills, and location preferences.

USER FLOW
1. User uploads CV or uses already parsed CV from Resuone
2. User selects:
   - preferred location
   - remote / hybrid / onsite
   - role type
   - salary expectation (optional)
3. System extracts candidate profile
4. System fetches live jobs from external APIs
5. System ranks the jobs against the candidate profile
6. User sees:
   - best-fit jobs
   - match score
   - why this job fits
   - what is missing
   - direct apply link

RECOMMENDED JOB DATA SOURCE
Primary: Adzuna API
Secondary/fallback: The Muse API

WHY
- Adzuna provides search endpoints and job categories
- The Muse provides public developer access for jobs and companies
- Build the source layer abstractly so more APIs can be added later

JOB PROVIDER LAYER
Create lib/jobs/providers/adzuna.ts

export async function fetchAdzunaJobs({
  country = "gb",
  page = 1,
  what,
  where,
}: {
  country?: string;
  page?: number;
  what: string;
  where?: string;
}) {
  const appId = process.env.ADZUNA_APP_ID!;
  const appKey = process.env.ADZUNA_APP_KEY!;

  const params = new URLSearchParams({
    app_id: appId,
    app_key: appKey,
    what,
    results_per_page: "20",
  });

  if (where) params.set("where", where);

  const url = `https://api.adzuna.com/v1/api/jobs/${country}/search/${page}?${params.toString()}`;
  const res = await fetch(url, { cache: "no-store" });

  if (!res.ok) throw new Error("Failed to fetch Adzuna jobs");

  return res.json();
}

Create lib/jobs/providers/muse.ts

export async function fetchMuseJobs({ page = 1, category }: { page?: number; category?: string }) {
  const apiKey = process.env.THE_MUSE_API_KEY!;
  const params = new URLSearchParams({
    page: String(page),
    api_key: apiKey,
  });

  if (category) params.set("category", category);

  const url = `https://www.themuse.com/api/public/jobs?${params.toString()}`;
  const res = await fetch(url, { cache: "no-store" });

  if (!res.ok) throw new Error("Failed to fetch The Muse jobs");

  return res.json();
}

    salary_max: job.salary_max || null,
    currency: "GBP",
    description: job.description || "",
    apply_url: job.redirect_url || "",
    source_payload: job
  };

export function normalizeMuseJob(job: any) {
  return {
    source: "themuse",
    external_job_id: String(job.id),
    title: job.name || "",
    company: job.company?.name || "",
    location: Array.isArray(job.locations) ? job.locations.map((x: any) => x.name).join(", ") : "",
    employment_type: job.type || "",
    remote_type: "",
    salary_min: null,
    salary_max: null,
    currency: "",
    description: job.contents || "",
    apply_url: job.refs?.landing_page || "",
    source_payload: job
  };
}

FETCH STRATEGY
For each candidate profile:
1. Take top 3 target_roles
2. Search jobs for each role
3. Search by preferred location if set
4. Also search remote if requested
5. Store all jobs in external_jobs_cache
6. Deduplicate by source + external_job_id

JOB RECOMMENDATION LOGIC
Do NOT send 200 jobs to Claude at once.
Use a 2-step ranking process:

STEP 1 — local pre-filter
Score quickly based on:
- keyword overlap
- skill overlap
- title overlap
- location preference match

STEP 2 — AI reranking
Take top 20 prefiltered jobs
Send compact candidate profile + compact job objects to Claude
Ask Claude to return top 10 ranked jobs with:
- match score
- reasons
- missing requirements
- summary

ADD LOCAL PREFILTER
Create lib/jobs/ranking.ts

function overlapCount(a: string[] = [], b: string[] = []) {
  const aa = new Set(a.map(x => x.toLowerCase()));
  const bb = new Set(b.map(x => x.toLowerCase()));
  let count = 0;
  for (const item of aa) {
    if (bb.has(item)) count++;
  }
  return count;
}

export function prefilterJobs(candidateProfile: any, jobs: any[]) {
  return jobs
    .map((job) => {
      const text = `${job.title} ${job.description} ${job.company} ${job.location}`.toLowerCase();

      const roleScore = (candidateProfile.target_roles || []).reduce((acc: number, role: string) => {
        return acc + (text.includes(role.toLowerCase()) ? 8 : 0);
      }, 0);

      const skillScore = (candidateProfile.core_skills || []).reduce((acc: number, skill: string) => {
        return acc + (text.includes(skill.toLowerCase()) ? 3 : 0);
      }, 0);

      const toolScore = (candidateProfile.tools || []).reduce((acc: number, tool: string) => {
        return acc + (text.includes(tool.toLowerCase()) ? 2 : 0);
      }, 0);

      const locationScore =
        candidateProfile.preferred_locations?.some((loc: string) =>
          text.includes(loc.toLowerCase())
        ) ? 6 : 0;

      return {
        ...job,
        pre_score: roleScore + skillScore + toolScore + locationScore
      };
    })
    .sort((a, b) => b.pre_score - a.pre_score);
}
